# 🎯 境界値分析 - 学習順序ガイド

## 📚 学習の全体像

### 🎯 学習目標
- **境界値分析の基本概念**を理解し、実務で効果的に適用できる
- **同値分割との組み合わせ**により、効率的で漏れの少ないテスト設計ができる
- **境界値における欠陥の特性**を理解し、重要な境界を特定できる

### 👥 対象者
- 同値分割の基礎を理解している方（必須前提）
- より精度の高いテスト設計技法を学習したい実務者
- 境界値における欠陥検出率を向上させたい方

### ⏱️ 推奨学習時間：**3-5時間**

## 📊 同値分割との関係性

### 🔗 **境界値分析の位置づけ**
```
ブラックボックステスト技法
├── 同値分割（基礎・必須前提）
├── 境界値分析（同値分割の強化拡張）
├── デシジョンテーブル（複合条件）
└── 状態遷移テスト（状態変化）
```

### 🎯 **なぜ境界値分析が重要か**
- **欠陥の偏在**：境界付近に欠陥が集中する統計的事実
- **効率的強化**：同値分割を少ないコストで大幅強化
- **実務必須性**：ほぼ全ての数値・範囲検証で必要
- **品質保証**：見落としがちな境界欠陥を確実に検出

### 🔄 **統合適用の価値**
同値分割 + 境界値分析により：
- **欠陥検出率 20-40% 向上**
- **テストケース増加は最小限**（10-20%程度）
- **論理的で説明しやすい設計**
- **レビューしやすい体系的構造**
